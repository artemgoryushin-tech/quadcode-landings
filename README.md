# Quadcode Landing Pages

Static landing pages for Quadcode's partner programs.

## Pages

### Creator Program (`/creator-program`)
Landing page for Quadcode's content creator partnership. Creators produce short-form video content about the white-label brokerage platform across TikTok, Reels, and YouTube Shorts, earning based on performance.

### Ambassador Program (`/ambassador-program`)
Landing page for Quadcode's ambassador partnership. Ambassadors represent the brand, drive awareness, and earn through a structured reward program.

## Structure

```
├── creator-program/
│   ├── index.html
│   └── assets/
└── ambassador-program/
    ├── index.html
    └── assets/
```

Each landing page is self-contained with its own `index.html` and `assets/` directory.

## Deployment

Each subdirectory can be deployed independently as a static site (Vercel, Netlify, GitHub Pages, etc.).
